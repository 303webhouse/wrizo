# Wrizo — North Star (v0.2)

## Promise
A writing studio that keeps you moving. It does not write for you.

## The wedge
The writing app that gets you writing — and can't be fidgeted with.

## Architecture (as-built)
- **Two Minds.** The **Middle Door** (emergence-first: drop into a fragment field,
  write from heat) and the **Trellis** (structure-first: theme → principles →
  points → write into slots) over ONE fragment substrate, traversed in opposite
  directions. (PHILOSOPHY.md.)
- **Drawers → Binders → Pages**, on the fragment substrate. A project is a Binder;
  its writing lives as **Pages** (manuscript chapters + support pages), never one
  body. A page's home is exactly one of Binder / Shelf / Journal.
- **Domain lives on the binder** — `type` (creative / academic / professional) ×
  `kind` (Book, Short fiction, Screenplay, Essay, Thesis, Paper, Article, Report,
  Proposal, …) — set at creation, **never** an app-level mode.
- **Modes on one surface.** *Free write* (forward-only generation, AI sealed) and
  *Draft* (free edit) are live; *Format / Workshop / Publish* are future.
- **Forward-only runway.** Backspace strikes, never erases; struck text stays
  visible and drops out of the prose. Shaping is a separate, later motion.
- **Chrome dissolve.** The interface recedes on the keystroke and returns at rest —
  decisions live at the edges, never the middle.
- **The gate.** The front door is a forced first-write: you're already writing
  before you're "in."

## North-star metric
**TTFK — time to first keystroke.** Instrumented per surface (page / journal /
sprint) with a Desk→ink funnel stamp (F5). The **F-arc** ("from open to flow,"
F1 resume → F6 first-line invitation) is its first optimization pass.

## Ink canon (2026-07-03)
**Ink is sealed in the Journal.** The Journal is the drawing pad + diary; **Pages
are the typewriter**; ink reaches a project only by **porting**. On every
non-Journal writing surface the stylus produces no ink, no characters, and no OS
handwriting recognition. (INK arc: I0 discipline → I1 render → I2 port → I3 unlink.)
