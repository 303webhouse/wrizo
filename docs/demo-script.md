# Demo Script (5-7 minutes)

## Goal
See whether the app reliably gets someone writing within 2 minutes and keeps them moving.

## Path A - Pantser (2 minutes)
1) Start Writing (Quick Sprint)
2) Write 3 lines (anything)
3) Optional: tap Nudge once
4) Finish -> Save as Project

Ask:
- Did anything slow you down before writing?
- Did Nudges help or distract?
- Did you feel pulled forward or managed?

## Path B - Plotter (3-5 minutes)
1) Create New Project -> Creative
2) Structure Wizard -> pick Story Circle
3) Start Writing -> save beat bullets
4) Structure Board -> Set as Next Beat

Ask:
- Did structure help or feel like homework?
- Did you always know what to do next?
- What tempted you into side-quests?

## Notes to record
- Time-to-first-words (seconds)
- Moments of confusion
- Any rage-quit points
- Feature requests (write verbatim)
